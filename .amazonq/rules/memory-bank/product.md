# McGarrah Technical Blog - Product Overview

## Project Purpose
Personal technical blog and website for Michael McGarrah, a technologist specializing in cloud architecture, machine learning, and enterprise infrastructure. The site serves as both a professional portfolio and knowledge sharing platform for technical content.

## Key Features

### Content Management
- **Jekyll-powered static site** with GitHub Pages hosting
- **Extensive blog archive** with 100+ technical articles spanning 2001-2026
- **Category and tag organization** for easy content discovery
- **Draft management system** for content development
- **RSS feed** for content syndication

### Technical Content Focus
- **Homelab infrastructure** - Proxmox, Ceph, virtualization
- **Networking** - SFP+, switches, VPN, SDN
- **Hardware projects** - Dell Wyse 3040, refurbished equipment
- **Software development** - Jekyll, Ruby, Python, automation
- **Cloud technologies** - AWS, containerization, DevOps

### User Experience
- **Responsive design** with mobile optimization
- **Search functionality** via Google Custom Search
- **Comment system** using Giscus (GitHub Discussions)
- **Reading time estimates** and excerpt previews
- **Mermaid diagram support** for technical illustrations
- **Code syntax highlighting** with copy buttons

### SEO and Analytics
- **Comprehensive SEO optimization** with structured data
- **Google Analytics** integration for traffic insights
- **Sitemap generation** for search engine indexing
- **Canonical URL management** for duplicate content prevention
- **Social media integration** with LinkedIn, GitHub, ORCID profiles

### Privacy and Compliance
- **GDPR compliance** with cookie consent management
- **Privacy policy** and transparent data handling
- **Ad-free experience** with optional AdSense integration

## Target Users

### Primary Audience
- **Technical professionals** seeking infrastructure and homelab guidance
- **System administrators** working with Proxmox, Ceph, and virtualization
- **DevOps engineers** interested in automation and best practices
- **Students and researchers** in computer science and networking

### Secondary Audience
- **Potential employers** reviewing technical expertise and communication skills
- **Open source contributors** interested in Jekyll plugins and automation
- **Hardware enthusiasts** working with refurbished enterprise equipment

## Value Proposition
Provides in-depth technical documentation and real-world implementation experiences that bridge the gap between enterprise technologies and homelab environments, offering practical insights for both learning and professional development.
